Help & Shortcuts

Quick overview of features and keyboard shortcuts.

Keyboard

  • Ctrl+K — Focus quick search (dashboard)
  • Ctrl+B — Focus book search (search books page)
  • Ctrl+M — Focus member search (search members page)
  • Esc — Close modals / dialogs

Features

  • Quick Actions panel on the dashboard for fast issue/return
  • Live partial search on books & members
  • Export books to CSV from View Books
  • Animated UI and toast notifications
